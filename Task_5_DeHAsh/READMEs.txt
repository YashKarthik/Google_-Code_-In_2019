Project description: A program that hashes plain text provided by the user in various formats and also performs a brute force 
attack using the list of password provided in the text file.

Includes: 1)The python script for the two functions.
          2)The text file containing a list of possible passwords.
          3)The screen recording of the script in execution.
          
The text file used for decryption is by default set to passW.txt
