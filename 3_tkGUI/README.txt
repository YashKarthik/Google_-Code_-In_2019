Steps for the python script:

Step_1: Import the necessary modules-tkinter, webbrowser, os, subprocess, askopenfile.
Step_2: Define the buttons and labels and bind them to their commands.
Step_3: Define the commands/functions bound to the buttons.
Step_4: Do the layout for the buttons and labels.

Steps for automated GUI launch:

Step_1: Use Automater in the Applications mode.
Step_2: Type in the terminal command for running the script.
Step_3: Save it as application.
Step_4: Go to system preferences and in user logins set login items as the automater file in applications folder


SCRIPT SPECIFIC FOR MAC OS ONLY!
